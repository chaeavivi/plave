# 추천템 (간단 버전) 🛍️

서버·DB·빌드 없이 **파일 하나(`index.html`)로 도는** 추천템 공유 페이지입니다.
링크만 붙여넣으면 이미지가 자동으로 채워지고, 카테고리 필터와 "사러갈 때 체크" 기능이 있어요.

## 1) 그냥 내 컴퓨터에서 보기
`index.html` 파일을 **더블클릭**하면 바로 열립니다. (설치 아무것도 필요 없음)
- 이 상태는 **"이 기기에만 저장"** 모드예요 → 내가 올린 건 내 브라우저에만 남습니다.

## 2) 인터넷 주소로 공개하기 (GitHub Pages, 무료)
1. GitHub 저장소에 이 `index.html`을 올립니다.
2. 저장소 **Settings → Pages** 로 이동
3. **Branch** 를 `main`(또는 파일이 있는 브랜치), 폴더를 `/root` 또는 `/docs` 로 지정 후 Save
4. 잠시 뒤 `https://<아이디>.github.io/<저장소>/` 주소가 생깁니다.
   - 이 `index.html`이 `simple/` 폴더에 있다면 주소 끝에 `/simple/` 을 붙이세요.

> GitHub Pages는 파일을 그대로 보여주기만 하므로, 위 1)과 똑같이 **각자 브라우저에 저장**됩니다.
> 여러 명이 **같은 목록을 공유**하려면 아래 3)을 설정하세요.

## 3) 여러 명이 함께 공유하기 (Firebase, 무료·5분)
`index.html` 안의 `firebaseConfig` 값만 채우면 공유 모드로 바뀝니다.

1. https://console.firebase.google.com 접속 (구글 로그인) → **프로젝트 만들기**
2. 왼쪽 **빌드 → Firestore Database → 데이터베이스 만들기** →
   위치 선택 후 **"테스트 모드로 시작"** 선택 (나중에 규칙 강화 가능)
3. 프로젝트 설정(⚙️) → **내 앱 → 웹 앱(</>) 추가** → 앱 등록
4. 화면에 나오는 `firebaseConfig` 의 `apiKey` / `authDomain` / `projectId` 값을
   `index.html` 상단 `firebaseConfig` 에 붙여넣기
5. 저장 후 새로고침 → 상단에 초록불 **"공유 중"** 표시가 뜨면 완료!

이제 이 페이지에 접속하는 모두가 같은 추천 목록을 보고 추가할 수 있어요.

### 참고
- 링크에서 이미지·제목 자동 가져오기는 무료 서비스(microlink.io)를 사용합니다.
  일부 사이트는 자동 추출이 안 될 수 있고, 그럴 땐 직접 입력하면 됩니다.
- "테스트 모드" Firestore는 30일 후 읽기/쓰기가 막힙니다. 계속 쓰려면
  Firestore **규칙(Rules)** 에서 접근을 허용하도록 바꿔주세요.
